# XFCE VNC Desktop for Render

One-file deployment of XFCE desktop to Render.com

## Quick Start
1. Copy `Dockerfile` to your repository
2. Deploy to Render.com
3. Access at: `https://your-service.onrender.com`
4. Password: `password123`

## Features
- XFCE Desktop Environment
- VNC access via browser
- Optimized for Render free tier
- Always-on with UptimeRobot
